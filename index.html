Bem-vindo às Atividades Web

Nesta aplicação, você encontrará duas atividades principais:

1. Gerenciador de Notas

No Notes, você pode gerenciar suas notas de forma prática e simples utilizando a API JSONPlaceholder. As funcionalidades incluem:

  • Criar novas notas
  • Editar notas existentes
  • Excluir notas
  • Buscar notas específicas

2. Batalha de Cartas da Marvel

Na Batalha de Cartas da Marvel, você pode explorar diversos personagens do universo Marvel, utilizando a API da Marvel. As funcionalidades incluem:

  • Buscar heróis por nome
  • Ver detalhes do herói selecionado
  • Selecionar até 5 heróis para cada jogador
  • Remover heróis selecionados
  • Realizar uma simples batalha entre os heróis selecionados pelos dois jogadores
